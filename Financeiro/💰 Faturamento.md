## 📌 Contexto

Tela acessada pelo menu [[🏦 Conta]] do módulo financeiro.

---

## 🧩 Campos do Formulário

### 🆔 Registro

- Tipo: string
    
- Obrigatório: ✔

📌 **Comportamento:**

- Campo inicia em branco
    
- Pode ser utilizado para:
    
    - Criar novo cadastro (preenchendo os dados)
        
    - Consultar registro existente (pressionando Enter)

---

### 🏥 Convênio

- Tipo: string

---

### 👨‍⚕️ Médico

- Tipo: string

---

## ⚙️ Ações da Tela

- Informar registro
    
- Consultar registro (via Enter)
    
- Preencher dados
    
- Salvar

---

## 🔁 Fluxo

1. Tela inicia com campo **Registro** em branco
    
2. Usuário informa o registro:
    
    - Pressiona Enter → consulta dados
        
    - Ou continua preenchendo para novo cadastro
    
3. Usuário preenche os demais campos
    
4. Salva o registro

---

## ⚠️ Regras de Comportamento

- Campo **Registro** é obrigatório
    
- A mesma tela permite:
    
    - Consulta
        
    - Cadastro
    
- Consulta é realizada ao pressionar **Enter**
