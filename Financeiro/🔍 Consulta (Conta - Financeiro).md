## 🔁 Fluxo

1. Tela inicia com campo de consulta em branco
    
2. Usuário informa o registro
    
3. Pressiona Enter para consultar
    
4. Sistema exibe os dados da conta

---

## ⚠️ Regras de Comportamento

- Consulta realizada ao pressionar **Enter**
    
- Tela apresenta os dados da conta faturada
